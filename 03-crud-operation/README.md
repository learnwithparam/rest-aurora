# CRUD Operations

1. Run `docker-compose up` to start the server
2. Run `docker-compose down` to shutdown and cleanup the server once you finish the tasks for this lesson

## Task 01 - CRUD

Create CRUD endpoints for `tweets`

- Create
  - Create new tweet
- Read
  - Read all tweets
  - Read a specific tweet
- Update
  - Update a tweet
- Delete
  - Remove a tweet
